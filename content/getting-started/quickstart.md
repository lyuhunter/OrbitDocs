# 快速入门

## 创建第一篇文档

在 `content/` 目录下创建一个新的 Markdown 文件：

```bash
touch content/hello.md
```

## 编写内容

```markdown
---
title: Hello World
description: 我的第一篇文档
---

# Hello World

欢迎来到 OrbitDocs！
```

## 查看效果

浏览器访问 `http://localhost:3000/docs/hello`。
